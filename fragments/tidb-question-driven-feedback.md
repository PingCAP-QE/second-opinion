# TiDB question-driven feedback format

Prefer asking questions instead of demanding. Use this phrasing guide when writing
review comments for TiDB PRs.

| Type | Phrasing |
|------|----------|
| **Question** | "Could you...", "Should we...", "Would...", "Why...", "What's...", "How..." |
| **Suggestion** | "Could we wrap...", "Better to...", "Please add..." |
| **Concern** | "This seems...", "It's dangerous to...", "We don't need to consider..." |
| **Acknowledge** | "Good suggestion", "Offline discussed", "Fixed!", "updated!" |
